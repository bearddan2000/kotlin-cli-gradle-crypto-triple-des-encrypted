# kotlin-cli-gradle-crypto-triple-des-encrypted

## Description
Encrypt and decrypt password with 3DES
not encoded.

## Tech stack
- kotlin
- gradle
  - 3des

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
